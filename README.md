# 🗺️ Campus Navigation System

A smart and interactive campus navigation system designed to help users easily locate buildings, rooms, and important places within a campus.

---

## 📌 Features

* 📍 Click-based location selection
* 🧭 Interactive map interface
* 🏫 Room-wise navigation system
* 🌙 Dark mode support
* 📱 Mobile responsive design
